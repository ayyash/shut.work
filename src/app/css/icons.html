Icons

Icons are set by the class .symbol, which depends on an iconated font, adding .icon-classname adds the character defined in CSS. Using open tools like icomoon and fontello, you can create the font, and just add new characters. Following are some of the existing icons defined in the font included in css folder (vpicons).

Ligatures are also supported, using two methods, using the i element, and adding text of the ligature, or by defining data-icon property to an element that has symbol class.

If you are using LTR and RTL, avoid using ligatures for icons that indicate direction (like arrow-right) and depend on CSS definition.

<div class="symbol" data-icon="plus">This adds a ligature icon for icon-plus</div>
<i class="large">add</i> This adds icon-plus as well. 
<span class="symbol icon-plus">This also places a plus to the left of the element</span>
