Width

Use the .c-x defined in grid, to define percentage widths. Best if you define width in colgroup, as follows

And .md-x, .lg-x also apply.

<table class="tabulated spacedout w100">
	<colgroup>
		<col class="c-1 md-3">
		<col class="c-2">
		<col class="c-4">
		<col class="c-1">
		<col>
	</colgroup>
	<tr>
		<th>c-1 md-3</th>
		<th>c-2</th>
		<th>c-4</th>
		<th>c-1</th>
		<th>*</th>
	</tr>
	<tr>
		<td>cell</td>
		<td>cell</td>
		<td>cell</td>
		<td>cell</td>
		<td>cell</td>
	</tr>
