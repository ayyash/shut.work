Stacks

Stacks are a way to layout the blocks quickly. The main two container types are .detailed and .overview. Inside of which, the .stack and .stacklist for an ol element, or a dl element, are expected to layout as following:

Both classes (stack stacklist) can be added together to the ol element to reduce markup.

.detailed stack
Detailed fieldset
  1. Stacklist item inside a stack div, for detailed stack
Detailed datalist
Stacked DD, no need for special classes 
<div class="detailed">
	<fieldset>
			<legend>Detailed fieldset</legend>
			<div class="stack">
				<ol class="stacklist">
					<li>
						Stacklist item inside a stack div, for detailed stack
					</li>
	 				<li>
	 					<label>
							A second stacklist item
						</label>
	 				<input type="text" />
				</li>
				</ol>
			</div>
		</fieldset>
		<dl>
			<dt>Detailed datalist</dt>
			<dd>
				Stacked DD, no need for special classes
			</dd>
		</dl>
</div>
